# Birthday Site

This is a single-page birthday experience with a countdown, memories, message, and surprise reveal.

## Quick edits
- Name: edit the `[Name]` text in `index.html`
- Countdown date: edit `targetDate` in `script.js`
- Memories and message: replace the text in `index.html`
- Music: drop an MP3 at `assets/song.mp3`
- Images: compress photos (300-600KB each) for faster loading on mobile

## Run locally
Open `index.html` in a browser.

## GitHub Pages
1. Create a GitHub repo and add these files.
2. Push to `main`.
3. In the repo settings, enable GitHub Pages for the `main` branch root.
4. Your site will publish in a few minutes.
